Simple Question Answer

1. What is DOM and what is the purpose of DOM?

Ans: Document Object Model (DOM) is the abbreviation for the programming interface for web content. It displays the nodes, or logical structure, of texts. Programmers can create and construct documents, navigate their structure, and add, alter, or delete elements and information using the Document Object Model. With a few exceptions, anything present in an HTML or XML document can be read, altered, removed, or added using the Document Object Model.

3. What is event bubble?

Ans: Event bubbling is a type of DOM event propagation.This particular event starts out by triggering on the deepest target element and progresses through the target element's parents in the same nesting hierarchy until it reaches the outermost DOM element or document object. It is one method the browser uses to handle events. Events are things the user does, like clicking a button or changing a field, for example. When a specific type of user interface event occurs, such as when a button has been clicked or a webpage has finished loading, code is executed using event handlers.

4. What is a callback function and why will you use it?

Ans: In Javascript, a function that receives an argument from another function is known as a callback function. It is then invoked within the parent function to finish a process or operation. To explain it more simply, it is a function that will be called (performed) after its parent function, the function into which it is supplied as an argument, has finished running.
